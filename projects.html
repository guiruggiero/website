guimail

JavaScript

AUTOMATION & HARDWARE

on-air-sign

Python

A Raspberry Pi Pico 2 W-powered NeoPixel ring that signals meeting and camera status for a home office — idle (off), in a meeting (yellow), or camera on (red). A Windows host monitor detects active conferencing apps across Zoom, Google Meet, Microsoft Teams, Slack Huddle, and Amazon Chime, then pushes state to the device. Launches automatically via Windows Task Scheduler.

Raspberry Pi Pico 2 W NeoPixel MicroPython Windows
guiruggiero/on-air-sign

Python

A collection of Python monitoring scripts built out of frustration with refreshing pages manually: an availability checker for the official Olympics merchandise store with email alerts, and appointment trackers for German government offices (Foreigners Office and Driver's License) that fire an email or SMS the moment a slot opens up.
